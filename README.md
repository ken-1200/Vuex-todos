# vuejs_todos

## Vuex-todos
Vue.js : Todoアプリ作成
 - Notion的な
   - 目標の登録
   - 目標にリンクした行動目標の登録
   - 目標・行動目標の一覧表示
   - 目標・行動目標の編集
 - コードの実行手順
   - Vuejsの環境準備
   `npm install -g @vue/cli`
   - projectの作成
   `vue create vuejs_todos`
   - Vuexのインストール
   `npm install vuex`
   - VueRouterのインストール
   `npm install vue-router`
   - Sassのインストール
   `npm install sass-loader node-sass`
 - 実行に必要な環境
   - vue: 2.6.11
   - vuex: 3.4.0 
   - vue-router: 3.3.4
